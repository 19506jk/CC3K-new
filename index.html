Instructions

  • To win the game you have to get to the 10th floor
  • To make a movement for the player, type the command and follow by enter

Move:

  • no = move north
  • ea = move east
  • so = move south
  • we = move west
  • nw = move northwest
  • ne = move northeast
  • se = move southeast
  • sw = move southwest

Attack:

  • input 'a' followed by direction
  • Ex: a no = attack north

Use potion:

  • input 'u' followed by direction
  • Ex: u no = use the potion at north
