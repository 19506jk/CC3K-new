CC3K-new
========

Written from Louis &amp; Danny's code

## Instructions

### Move
- no = move north
- ea = move east
- so = move south
- we = move west
- nw = move northwest
- ne = move northeast
- se = move southeast
- sw = move southwest

### Attack
input 'a' followed by direction <br>
Example: a no = attack north

### Use potion
input 'u' followed by direction <br>
Example: u no = use the potion at north

### Visit town
input "town"
